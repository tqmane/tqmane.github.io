Build device tree

DumperX makes it easy to build a device tree.

To build Device tree:

git clone https://github.com/DumprX/DumprX.git
./dumper.sh 'Firmware File/Extracted Folder -OR- Supported Website Link'

Like this:

./dumper.sh SAMFW.COM_SC-03L_DCM_SC03LOMU1DWC1_fac.zip

Once the dump is done, you can find the twrp device tree at <working folder>/DumperX/out/twrp-device-tree/<device code name>/.
